## 📋 Sobre o Projeto
Este projeto consiste no desenvolvimento de um sistema web de gestão educacional e administrativa para a ONG Ábaco. O objetivo é digitalizar e otimizar os processos de cursos profissionalizantes gratuitos oferecidos à comunidade.

## 🚧 Status do Projeto
⚠️ **Em Desenvolvimento Ativo**
Este projeto está em fase inicial de desenvolvimento. Funcionalidades podem estar incompletas ou sujeitas a mudanças significativas.

## ✨ Funcionalidades Planejadas
- Gerenciamento de Alunos e Matrículas
- Controle de Frequência e Avaliações
- Gestão de Materiais e Estoque
- Dashboards de Acompanhamento Administrativo

## 📦 Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
- [Git](https://git-scm.com)
- [Node.js](https://nodejs.org/en/)
- [Docker](https://www.docker.com/) e [Docker Compose](https://docs.docker.com/compose/)

## 🔧 Instalação e Configuração
```bash
# Clone este repositório
git clone [url-do-projeto]

# Acesse a pasta do projeto no terminal/cmd
cd sga-abaco

# Suba os containers do Docker
docker-compose up --build
```
